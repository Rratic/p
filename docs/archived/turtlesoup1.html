    海龟汤归档

    A

    A 邀请 B、C 参加一次聚会，但是 B 拒绝了，而 C 半路有事没有去，甚至 A 自己也没有去。 一个月后 B 邀请 A、C 参加一次聚会，但是 A 联系不到，而 C 拒绝了。 一个月后有人发现 A、B、C 都呈死亡状态。

    A、B 与 C 使用了人工智能系统替代自己参与社交，这包含了在死后假装自己还活着的功能。

    Note

    参考了一篇科幻。名字忘了

    B

    关于碘酸铊，某校某化学实验社社长（以下记作 A），的不同死法

    B1

    社团节来临之际，A 死在了实验室里，一些东西被撞翻了。

    A 想测试新的项目：倒着做实验，在拿试管时习惯性地用手正握，于是寄了。因挣扎从天花板上摔下来时把一些东西撞翻了。

    B2

    A 死在了实验室里，手上有伤口。

    B 利用蜜蜂蛰咬制造了伤口，A 试图用实验室中药剂中和，但药剂被 B 提前换了。

    B3

    A 将一些蓝色液体倒进试管里，发现变成了绿色，大惊失色，跑出了实验室，后来死了。

    B 在水管管理的操作间内试图投毒，因机缘巧合让氯化钠进入了实验室水管中，A 清洗试管后它沾上了氯化钠溶液，与硫酸铜络合。 A 意识到这一问题，跑到操作间查看，目睹了 B 的所为，于是被 B 解决了。
